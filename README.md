# الدليل الإرشادي: مواجهة هلوسة الذكاء الاصطناعي وحماية النزاهة القانونية

## نظرة عامة
هذا الدليل موجه للمحامين والمختصين القانونيين الذين يستخدمون أدوات الذكاء الاصطناعي (مثل ChatGPT) في أعمالهم. يوثق الدليل حالات حقيقية صدرت فيها عقوبات قضائية بسبب الاعتماد على مخرجات الذكاء الاصطناعي دون تحقق، ويقدم أدوات عملية لتجنب المخاطر.

## المحتويات
- **500+ حالة موثقة** من الولايات المتحدة
- **جدول عقوبات مقارن** لـ 6 دول (الولايات المتحدة، كندا، أستراليا، الهند، إسرائيل، الأردن)
- **بروتوكول التحقق رباعي الخطوات** للكشف عن الاستشهادات المختلقة
- **إطار PCS‑Legal** لصياغة أوامر الذكاء الاصطناعي بشكل آمن
- **قائمة تحقق تفاعلية** قبل تقديم أي مستند للمحكمة
- **أسئلة شائعة** حول المسؤولية القانونية والإفصاح

## التقنيات المستخدمة
- HTML5 / CSS3 / JavaScript (纯前端، لا يحتاج خادم)
- دعم كامل للغتين العربية (RTL) والإنجليزية (LTR)
- تصميم متجاوب مع الهواتف والحواسيب اللوحية

## كيف تستخدم الدليل؟
1. افتح الصفحة في المتصفح.
2. استخدم زر تبديل اللغة (العربية/English) في الأعلى.
3. تصفح الحالات الموثقة، جدول العقوبات، وقائمة التحقق التفاعلية.
4. أكمل عناصر قائمة التحقق قبل تقديم أي عمل قانوني يعتمد على الذكاء الاصطناعي.
